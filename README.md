# FoodieGo - Food Delivery Application

## Project Overview

FoodieGo is a Java Full Stack Food Delivery Application developed using:

* Java
* JSP
* Servlets
* JDBC
* MySQL
* HTML
* CSS

## Features

* User Registration
* User Login
* Restaurant Listing
* Search Restaurants
* Menu Management
* Add To Cart
* Checkout
* Order Placement
* Order History
* Profile Management
* Offers Page
* Contact Page

## Database Tables

* User
* Restaurant
* Menu
* OrderTable
* OrderItem

## Technologies Used

Frontend:

* HTML
* CSS
* JSP

Backend:

* Java
* Servlets
* JDBC

Database:

* MySQL

Server:

* Apache Tomcat

## Project Flow

Register
→ Login
→ Browse Restaurants
→ View Menu
→ Add To Cart
→ Checkout
